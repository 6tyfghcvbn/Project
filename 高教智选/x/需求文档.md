# 需求文档

# 系统架构
本系统采用分层架构，包括前端、后端、数据库等。
前端采用Vue.js框架，后端采用Spring Boot框架，数据库采用MySQL。
例外本系统还包括爬虫模块，用于从互联网上获取数据。
爬虫模块采用Python语言编写，使用Scrapy框架。
数据来源：https://higher.smartedu.cn/courses
获取数据内容有：
1. 课程链接
2. 课程名称
3. 课程来源学校
4. 课程授课教师
5. 累计选课人次
要求：
1. 数据来源必须合法，不能包含任何违法、违规、侵权的内容。
2. 数据来源必须是公开的，不能是私有的。
3. 数据必须是多样的，不能只有一种数据。
4. 爬取的数据进行清洗和处理，确保数据的质量和一致性。
5. 爬取的数据必须是实时的，不能是历史的。
